1. Hvað gera eftirfarandi Linux skipanir?
	cd
	cd ~
	ls
	pwd
	mkdir

cd - change directory
cd ~ - fer heim 
ls - sýnir skrár á svæðinu sem þú ert á
pwd - print working directory
mkdir - make directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
	git log
	git status
	git diff
	git checkout

git clone - creates a working copy of a local repository
git log - sýnir hvaða git commit þú ert búinn að gera
git status - sýinir working tree status
git diff - sýnir þér muninn á eitthverju
git checkout - Updates files in the working tree to match the version in the index

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2
diff er almenna skipunin, diff --staged sýnir staged area en ekki working, git commit1 commit 2 ber saman
